C1 Exam Prep Landing Page

A high-converting landing page for a Polish language tutor specializing in C1 exam preparation.

Project Overview

This landing page is designed to drive student sign-ups for C1 exam preparation. It clearly communicates the value proposition, presents multiple pricing tiers, showcases social proof, and integrates a direct sign-up form for lead capture.

Technologies Used

HTML5 Tailwind CSS JavaScript Telegram Bot API (form handling)

Key Features

  • Hero section with strong headline and dual CTA buttons
  • Clear pricing packages and single-lesson options
  • Social proof via student results and testimonials
  • Sign-up form integrated with Telegram for instant notifications
  • Fully responsive design with fast load times
  • Clean visual hierarchy and distraction-free layout

My Role

I conducted target audience research, structured the content, wrote compelling copy, designed the layout, implemented the site with Tailwind CSS, integrated the sign-up form, and optimized for conversions with strategic CTAs and trust-building elements.
